# Place your job description documents here

Supported formats:
- PDF (.pdf)
- Word Documents (.docx, .doc)
- PowerPoint Presentations (.pptx, .ppt)

You can have multiple team JDs in a single document.
Just make sure each team section has a clear team name/identifier.

Example format:
  Team: Engineering
  Required Skills: Python, AWS, React
  Experience: 5+ years
  
  Team: Data Science
  Required Skills: Python, ML, SQL
  Experience: 3+ years
